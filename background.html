6. Cores de texto e planos de fundo

Cores de texto. Pode ser alterada via css, com o comando “color”, exemplo: 

 
	<p id=”teste”> Texto desejado <p>

	p#teste {
		color: red;
	}

Algo a se acrescentar são as mudanças de estilo de texto, onde podemos pegar fontes disponibilizadas pelo google fonts e importar para nosso site, com um simples passo-a-passo clicando aqui

6.2 Planos de fundo

São utilizados para modificar o fundo de algum objeto no html, podendo alterar a cor, tamanho, posição entre outras funções. O “background” que é utilizado pelo css possui várias derivações, em que as principais são: 


	div {
		background-color: red;
		background-position: right;
		background-image: url(“imagem.png”);
		background-size: cover;
	}

para mais detalhes sobre cada função, clique aqui.
