10. Formulários

Todo formulário deve estar entre a tag <form> , e dentro desta tag o atributo "type" muda a tag em alguns aspectos consideráveis. A seguir alguns exemplos:

Exemplo 1 

	
	<form>
		<label for="fname"> teste </label> 
		<input type="text" id="fname">
	</form>

RESULTADO

obs* note que ao clicar na palavra seleciona a caixa de texto também.
Isso é possível devido ao label, que está configurado para acessar o input ao clicar na palavra "teste", pois alguns usuário em mobile possuem os dedos muito grandes para a caixa de texto pequena, isso melhorar a experiência do usuário.

Exemplo 2 

 
	<form> 
		<label for="fnumber"> Digite um NÚMERO: </label>
		<input type="number" id="fnumber" >
	</form>

RESULTADO

Exemplo 3 

 
	<form> 
		<label for="imale"> Masculino </label>
		<input type="radio" id="imale" name="nradio" value="male" checked> 

		<label> Feminino </label>
		<input type="radio" id="ifem" name="nradio" value="fem">
	</form>

RESULTADO

10.1 Select e Option

A tag select te dará várias opções para escolha, e todas as escolhas são chamadas de options.
Exemplos: 

 
	<select name="estado" id="estado"> 
		<optgroup text-align="justify" label="Estado">
			<option> Guaramirim </option>
			<option> Jaraguá do sul </option>
			<option> Joinville </option>
			<option> Massaranduba </option>
			<option> São joão do itaperiú </option>
			<option> Blumenau </option>
		</optgroup>
	</select>
