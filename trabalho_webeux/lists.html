9. Listas

No html existem dois tipos de listas,:

Ordered lists (listas ordenadas) = ol

Unordered lists (listas não ordenadas) = ul

É importante resaltar que as listas devem estar dentro da tag < nav >, além disso, dentro das listas existem os itens (< li >).

Listas ordenadas 

 
	
	<nav>
	<h3>Resultado</h3>
	<ol>
		<li> Home </li>
		<li> Especificações </li>
		<li> Fotos </li>
		<li> Mutimídia </li>
		<li> Fale conosco </li>
	</ol>
	</nav>

Resultado

  1. Home
  2. Especificações
  3. Fotos
  4. Mutimídia
  5. Fale conosco

Listas com letras

podemos modificar as listas com o atríbuto "type", exemplo: 

 

<nav>
	<h1>Menu Principal</h1>
	<ol type="a">
		<li> Home </li>
		<li> Especificações </li>
		<li> Fotos </li>
		<li> Mutimídia </li>
		<li> Fale conosco </li>
	</ol>
</nav>

Resultado

  1. Home
  2. Especificações
  3. Fotos
  4. Mutimídia
  5. Fale conosco

ops* caso coloque em "A" na tag type, as letras ficarão maiúsculas. 

 
	
 Com números romanos: 


	<nav>
		<h1>Menu Principal</h1>
		<ol type="i">
			<li> Home </li>
			<li> Especificações </li>
			<li> Fotos </li>
			<li> Multimídia </li>
			<li> Fale conosco </li>
		</ol>
	</nav>

Resultado

  1. Home
  2. Especificações
  3. Fotos
  4. Multimídia
  5. Fale conosco
Caso for com "I", os numeros romanos serão em letras maiusculas também. 
 
	
Caso eu queria uma lista ordenada começando do numero 3 

	<nav>
		<h1>Menu Principal</h1>
		<ol type="1" start="3">
			<li> Home </li>
			<li> Especificações </li>
			<li> Fotos </li>
			<li> Mutimídia </li>
			<li> Fale conosco </li>
		</ol>
	</nav>

Resultado

  1. Home
  2. Especificações
  3. Fotos
  4. Mutimídia
  5. Fale conosco

funciona do mesmo jeito com as letras, onde o atríbuto "start" significa posição da letra, ou seja, start="3" == C

LISTAS NÃO ORDENADAS

São listas com símbolos, sem ordem númerica, alfabética ou romana.
Exemplo: 

 

	<nav>
		<ul>
			<li> Home </li>
			<li> Especificações </li>
			<li> Fotos </li>
			<li> Mutimídia </li>
			<li> Fale conosco </li>
		</ul>
	</nav>
Resultado :
  • Home
  • Especificações
  • Fotos
  • Mutimídia
  • Fale conosco

    valores para o type de UL

  • square: quadrados
  • circle: circulo com fundo branco
  • disc: circulo todo preto

LISTAS DENTRO DE LISTAS

Podemos fazer listas ul dentro de ol ou vice-versa. Ex: 

 
	<nav>
		<ul type="a">   // Lista não ordenada
			<li> Home </li>
			<li> Especificações </li>
			<li> Fotos </li>
				<ol>       // Lista ordenada
				         <li>músicas</li>	
				         <li>videos</li>
				</ol>
			<li> Mutimídia </li>
			<li> Fale conosco </li>
		</ul>
	</nav>
Resultado :

  • Home
  • Especificações
  • Fotos
    1. músicas
    2. videos
  • Mutimídia
  • Fale conosco

Atríbuto 'type' pode ser aplicados da mesma maneira.
